# ⚙️ **Installation**

### 1. Clone the repo

```bash
git clone https://github.com/ntdai95/ntdai95.github.io
cd ntdai95.github.io
```

### 2. Install dependencies

If `npm` is not installed, download Node.js from https://nodejs.org

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

Then open:

    http://localhost:5173/

---

## 🚀 **Deployment Instructions**

### 1. Make changes

Edit your source files (e.g. `.jsx`).

### 2. Publish to GitHub

```bash
publish
```

### 3. Automatic deployment

GitHub Actions will deploy live at:

    https://ntdai95.github.io
